# Commands

### Dev

```bash
yarn dev
```

### Docs dev

```bash
yarn docs:dev
```

### Lint

```bash
yarn lint
```

### Lint fix

```bash
yarn lint:fix
```

### Test

```bash
yarn test
```

### Database

```bash
    # initialize migrations
    ./bin/migrator init

    # create a new migration
    ./bin/migrator make <migration-name>

    # run migrations
    ./bin/migrator execute

    # update type schema
    ./bin/generate-schemas
```
